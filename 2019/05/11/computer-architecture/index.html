Computer Architecture

What is Computer Architecture

Classical definition

the attributes of a computing system as seen by the assembly language programmer,i.e.,the conceptual structure and functional behavior, as distinct from the logic design,and the physical implementation.â€”Amadahl,Blaaw,and Brooks,1964

attributes here are like:

  • data representation
  • addressing rules
  • registers definition
  • instruction set
  • interrupt system
  • working state definition and switching
  • memory system
  • message protection
  • I/O structure

Essence

a layer between SW and HW.

  • logical implementation:computer organization
  • physical implementation:computer implementation

an Architecture can have several organization,or physical implementation.

Broadest definition

the design of the abstraction layers that allow us to implement information processing applications efficiently using available manufacturing technologies.

Summary

Computer Architecture is an integrated Approach,what really matters is the functioning of the complete system,and Computer Architecture is not just about transisstors, individual instructions, or particluar implementations.
